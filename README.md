# QuadDesk

QuadDesk is a clean, efficient, and comprehensive desktop productivity application built entirely using **Python**. It brings all your essential daily tools together into a single, cohesive workspace.

## 🚀 Key Features

* **📅 To-Do List:** Plan your day, manage tasks efficiently, and check them off as you complete them.
* **⏱️ Stopwatch:** Track your time accurately for study sessions, work tasks, or productivity sprints.
* **🔢 Calculator:** A built-in, reliable calculator for quick everyday mathematical calculations.
* **🕒 Analog Clock:** A classic, clean visual timepiece integrated directly into the interface to keep you aware of the time.

## 🛠️ Built With

* **Python:** Core programming language.
* **Tkinter / CustomTkinter:** For building the graphical user interface (GUI).

2. Download and extract the `QuadDesk v2.0.zip` file.
